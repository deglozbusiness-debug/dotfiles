# dotfiles

Personal configuration files for my Linux setup.

## System
- **OS:** CachyOS (Arch-based)
- **WM:** Hyprland 0.55
- **Bar:** Waybar
- **Terminal:** Alacritty

## What's included
- `hypr/` — Hyprland window manager config (monitors, keybindings, autostart, rules)
- `waybar/` — Status bar config and styles

## Key bindings
| Key | Action |
|-----|--------|
| `Super + Q` | Close window |
| `Super + Enter` | Open terminal |
| `Super + Shift + W` | Open Typora |
| `Super + Ctrl + W` | Wifi controls |
